# homework-checker

## Description
Simple role-based web app where teachers and class monitors assign tasks and students mark completion

## Technologies Used
- PHP
- MySQL
- HTML/CSS
- JavaScript
- XAMPP

## Setup Instructions
1. Install XAMPP
2. Clone this repository to `htdocs` folder
3. Import `sql/database.sql` to phpMyAdmin
4. Configure database connection in `config.php`
5. Access via `https://localhost/index.php`

## Database Setup
- Database name: `ictsba`
- Import the SQL file from `/sql` folder
